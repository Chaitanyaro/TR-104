Training Overview

The main focus of this six months industrial training was to gain hands-on experience in Data Analytics and Business Intelligence by designing and developing an interactive OTT platform dashboard using Microsoft Power BI.

The project involved collecting, cleaning, and analyzing content data from Netflix, Amazon Prime Video, and Hotstar. The final outcome was a dynamic Power BI dashboard showcasing content distribution, genre trends, release year analysis, and platform-wise comparisons using interactive visuals, slicers, and drill-down features.

Objectives of Training

  • To understand the role and responsibilities of a Data Analyst.
  • To learn data collection, cleaning, and preparation techniques.
  • To analyze OTT platform content using Excel and Power BI.
  • To build interactive dashboards with filters and drill-downs.
  • To gain practical exposure to data-driven decision-making.

Daily Diary Entries

Day 1 – Introduction to Data Analysis

Learned the fundamentals of data analysis and the role of a data analyst. Understood how analysts collect, clean, and interpret data to convert raw information into meaningful insights using tools like Excel, SQL, Python, and Power BI.

Day 2 – Importance of Data Analysis & Industry Applications

Studied how data analysis supports evidence-based decision-making across industries such as healthcare, finance, retail, and entertainment. Discussed Netflix as a real-world example of data-driven content strategy.

Day 3 – Data-Driven Decision Making

Learned the complete data-driven decision-making cycle from defining objectives to evaluating results. Analyzed case studies of Netflix, Uber, and Starbucks to understand practical implementation.

Day 4 – Data Collection & Preparation

Gained knowledge of structured, semi-structured, and unstructured data. Learned data collection sources, ETL concepts, and data cleaning techniques using Excel and Power Query.

Remaining daily entries (Day 5 – Day 180) follow a similar structured format covering Python basics, data analysis concepts, Excel, Power BI dashboard development, data modeling, visualization, and final project implementation.

Final Project – OTT Platform Dashboard

The final project involved building an interactive Power BI dashboard comparing Netflix, Amazon Prime Video, and Hotstar based on content type, genre distribution, release year trends, and platform-wise availability. Data was collected and cleaned using Excel, followed by modeling and visualization in Power BI.

The dashboard included slicers, charts, KPIs, and drill-down features to allow users to explore trends easily and derive meaningful insights for decision-making.
