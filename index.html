Training Overview

The primary objective of this six months industrial training was to gain comprehensive hands-on exposure to Data Analytics and Business Intelligence by developing an interactive dashboard using Microsoft Power BI.

The project focused on analyzing content datasets from Netflix, Amazon Prime Video, and Hotstar to understand content trends, genre distributions, and release patterns.

Objectives of Training

  • Understand the fundamentals of data analytics and BI tools.
  • Learn data cleaning, transformation, and modeling techniques.
  • Analyze OTT content datasets using Excel and Power BI.
  • Design interactive dashboards for business insights.
  • Apply analytical thinking to real-world datasets.

Daily Diary Entries (180 Days Summary)

Days 1–15 : Introduction to Data Analytics ⌄

This phase introduced the fundamentals of data analytics, analytical thinking, and the responsibilities of a data analyst in modern organizations.

Case studies of Netflix and other OTT platforms demonstrated how data-driven decisions influence content recommendations and user engagement.

Days 16–30 : Data Collection & Preparation ⌄

Learned structured and unstructured data concepts along with real-world datasets related to OTT platforms.

Data cleaning techniques using Excel ensured high-quality datasets for further analysis.

Days 31–60 : Python for Data Analysis ⌄

Python programming basics, control structures, and data structures were covered to strengthen analytical logic.

Python was applied to small datasets to understand preprocessing workflows.

Days 61–90 : Excel & Pivot Analysis ⌄

Advanced Excel functions and Pivot Tables were used to analyze genre and platform-based content trends.

Days 91–150 : Power BI Dashboard Development ⌄

Power BI was used for data modeling, DAX calculations, and creating interactive visuals.

Dashboards were designed to compare Netflix, Prime Video, and Hotstar.

Days 151–180 : Final Project & Documentation ⌄

Final refinement, validation of insights, and documentation of the OTT dashboard project were completed.

This phase concluded the six months industrial training successfully.
