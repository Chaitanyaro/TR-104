Daily Diary Entries (180 Days Overview)

Days 1–15 : Introduction to Data Analytics & Decision Making

During the initial phase of training, the focus was on understanding the fundamentals of data analytics and the professional role of a data analyst. The learning included the responsibilities of data collection, cleaning, analysis, and interpretation to support business decisions.

Special emphasis was placed on how organizations like Netflix and Amazon Prime use data to understand customer behavior, content performance, and market trends. Various real-world case studies were discussed to understand data-driven decision-making.

By the end of this phase, a strong conceptual foundation was built regarding analytical thinking, problem identification, and the importance of accurate data.

Days 16–30 : Data Collection, Types of Data & Preparation

This phase focused on understanding structured, semi-structured, and unstructured data. Practical examples included CSV files, Excel sheets, JSON datasets, and OTT platform content metadata.

Learned multiple data collection methods such as surveys, web data extraction, and open datasets. Data preparation techniques like removing duplicates, handling missing values, and standardizing formats were practiced using Excel.

These skills later became essential for preparing Netflix, Prime Video, and Hotstar datasets before importing them into Power BI.

Days 31–60 : Python Basics for Data Analysis

This phase introduced Python programming fundamentals including syntax, variables, data types, conditional statements, loops, and functions. Practical exercises were performed to strengthen logical thinking.

Concepts such as lists, dictionaries, tuples, sets, and list comprehensions were covered to understand data structures used in analytics workflows.

Python was used for small-scale data manipulation tasks, reinforcing how scripting supports automation in data analysis projects.

Days 61–90 : Excel for Data Analysis & Reporting

In this phase, Excel was used extensively for data analysis. Functions such as VLOOKUP, XLOOKUP, IF, COUNTIFS, SUMIFS, and TEXT functions were practiced.

Pivot Tables and Pivot Charts were created to summarize large datasets and identify trends in OTT content such as genre distribution and release years.

Excel acted as the primary preprocessing tool before importing refined data into Power BI for dashboard creation.

Days 91–120 : Power BI Fundamentals & Data Modeling

This phase marked the transition to Power BI. Learned Power BI interface, data import methods, relationships, and data modeling concepts.

Multiple datasets from Netflix, Amazon Prime, and Hotstar were connected and transformed using Power Query Editor.

Basic DAX functions were introduced to create calculated columns and measures required for meaningful analysis.

Days 121–150 : Dashboard Design & Visualization

This phase focused on designing interactive dashboards using charts, cards, slicers, and filters. Best practices for layout, color selection, and readability were followed.

Separate visuals were created for content type comparison, genre-wise distribution, and release year trends across platforms.

Drill-down and cross-filtering features were implemented to improve user interaction and analytical depth.

Days 151–180 : Final Project Implementation & Review

The final phase involved refining the OTT dashboard, validating insights, and optimizing performance. Multiple revisions were made based on feedback.

The completed dashboard effectively compared Netflix, Amazon Prime Video, and Hotstar using interactive visual analytics.

This phase concluded with documentation, project presentation, and a review of skills gained during the six months industrial training.
